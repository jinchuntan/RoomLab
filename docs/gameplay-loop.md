# Gameplay Loop

RoomLab’s lesson loop is intentionally short, repeatable, and collaborative.

## High-level loop

1. Form room
2. Align nearby users
3. Spawn molecule challenge
4. Collaboratively place atoms
5. Reveal molecule fact and score
6. Run synchronized quiz
7. Advance to next molecule or reset

## First playable lesson: Molecule Builder

Included starter molecules:

- H2O
- CO2
- CH4

Each challenge includes:

- tray atoms
- expected slot layout
- target geometry
- quick fact
- real-world context
- three quiz prompts

## Per-round structure

### Build phase

- players grab atoms from the tray
- the shared board shows expected bonding slots
- valid placements snap to slots
- invalid placements are rejected

### Completion phase

- correct assembly triggers a celebration state
- team score increases
- molecule name and quick fact appear

### Quiz phase

- host starts the synchronized quiz
- players answer one question each in shared sequence
- geometry, bond count, and usage are reinforced

### Transition phase

- when the quiz completes, the host can spawn the next challenge
- after the final challenge, the room enters the completed state

## Educational intent

- water introduces bent geometry in a familiar context
- carbon dioxide contrasts with a linear structure
- methane adds a more spatially distributed tetrahedral arrangement

The order helps learners build intuition rather than jumping immediately into harder geometry.

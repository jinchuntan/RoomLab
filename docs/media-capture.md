# Media Capture Guide

Use this checklist to capture polished media for a README, hackathon submission, or application form.

## Recommended stills

1. Lobby state with host and guest connected, both marked ready.
2. Alignment step showing shared-origin confirmation and participant confidence.
3. Molecule assembly in progress with one player holding an atom and another seeing the synchronized result.
4. Completed molecule card with quick fact, score update, and quiz prompt.
5. Quiz round with synchronized answers and the shared scoreboard visible.

## Recommended GIFs / short clips

1. Host creates a room, guest joins from a second browser tab, and both align into the same shared scene.
2. One player drags an atom into a valid slot and the second player sees the snap in real time.
3. Team completes a molecule, receives the learning fact, and advances into the quiz.

## Capture tips

- Use two browser windows side by side to make the multiplayer sync immediately legible.
- Keep the debug diagnostics panel visible in at least one shot to reinforce technical credibility.
- If you record on Spectacles hardware later, mirror the same flow so the browser simulator and hardware demo tell the same story.

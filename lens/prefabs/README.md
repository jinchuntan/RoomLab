# Prefab Planning

Primitive-friendly prefab suggestions for a hardware implementation:

- `AtomOrb`: sphere mesh, label billboard, ownership highlight ring.
- `BondSlotGhost`: translucent placeholder showing expected atom type.
- `RoundCompleteBurst`: brief celebratory animation triggered after correct assembly.
- `ParticipantBadge`: nameplate and alignment confidence indicator.

The browser simulator already uses simple primitives and labels, so these prefabs can mirror that interaction language when moved into Lens Studio.

# Asset Placeholder Notes

RoomLab deliberately avoids proprietary assets.

Suggested starter assets for a Lens Studio implementation:

- Flat-shaded sphere materials for `H`, `O`, `C`, and `N`.
- Minimal icon set for ready, aligned, holding, and disconnected states.
- One neutral panel material for world-space instructional UI.
- Lightweight celebratory pulse or particle texture for round completion.

All of these can be recreated with basic primitives and procedural materials if no external asset pipeline is available.
